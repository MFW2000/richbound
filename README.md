# richbound

Richbound is a retro-style, text-based RPG game built entirely in C#, made to run in the terminal for Windows and Linux.

Set in modern times, you start from nothing and work your way to the top through legal or illegal means in a city full
of opportunities and dangers. Survive through begging, odd jobs, and petty crime - or build your own business empire.
The choice is yours.

## Features

Richbound offers a variety of features:

- Open-ended gameplay with multiple paths to success
- Text-based interface for a retro gaming experience
- Manage health, hunger and sleep to survive
- Manage your finances, inventory and assets
- Variety of activities including begging, odd jobs, petty crime
- Purchase and manage businesses with various available upgrades and expansions
- Purchase and use items and weapons to aid in your survival and success
- Purchase and use vehicles to improve travel and efficiency

## Running Requirements

To run the application, you need to following software for your operating system:

### Windows

- [.NET 8.0 Runtime](https://dotnet.microsoft.com/en-us/download/dotnet/8.0/runtime)

### Linux

- [.NET 8.0 Runtime](https://dotnet.microsoft.com/en-us/download/dotnet/8.0/runtime)

Additionally, the provided file must be made executable using the `chmod +x <filename>` command in the same directory.
